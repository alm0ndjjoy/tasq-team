Task Manager Project

In this project proposed by Generation USA to the students of the Junior Web Developer course, we created a Task Planner web app from scratch. The goal of this group-work is to give the students the opportunity to practice technical and non-technical skills. The project was done in 3 sprints of approximately 10 hours each.
